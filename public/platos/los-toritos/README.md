# Imágenes de Los Toritos

Esta carpeta contiene las imágenes específicas del menú de Los Toritos.

## Imagen del Calzone Primavera

Para agregar la imagen del calzone Primavera:

1. Guarda la imagen como: `calzone-primavera.jpg`
2. Colócala en esta carpeta: `public/platos/los-toritos/`
3. La ruta completa será: `/platos/los-toritos/calzone-primavera.jpg`

## Formato recomendado

- **Formato**: JPG o PNG
- **Tamaño**: 800x600px o similar (proporción 4:3)
- **Peso**: Máximo 500KB para mejor rendimiento
- **Nombre**: `calzone-primavera.jpg`

La imagen se mostrará:
- Como miniatura (40x40px) en la lista de items
- En tamaño grande (192px altura) en el modal al hacer click
